# Blender-Assisted Photogrammetry: Automating 3D Asset Creation Through Integrated Computer Vision Workflows

## Acknowledgements

This project builds upon several outstanding open-source tools and research frameworks from the photogrammetry, computer vision, and 3D graphics communities:

* [Blender](https://www.blender.org?utm_source=chatgpt.com) for procedural asset creation, geometry processing, rendering, and pipeline integration.
* [OpenCV](https://opencv.org?utm_source=chatgpt.com)  for computer vision operations including feature detection, calibration, and image preprocessing.
* [Open3D](https://www.open3d.org?utm_source=chatgpt.com)  for 3D data processing, mesh manipulation, point cloud operations, and visualization tooling. [open3d.org]
